About Me
A bit about how I work.

I'm an Associate Software Engineer building enterprise fintech applications — the kind where a misplaced decimal or a missing column breaks something real. My days are spent in Spring Boot services, Hibernate mappings, and Oracle SQL, tracing bugs down to the exact query that caused them.

I'm not the person who builds REST APIs from scratch — I'm the one who integrates and hardens them, closing the gaps and making sure they hold up in production. I've picked up frontend work along the way too, with Vaadin, React, and the occasional React Native fix.

Outside of feature work, I care about writing code I could hand to someone else and trust they'd understand it — no tribal knowledge required.
